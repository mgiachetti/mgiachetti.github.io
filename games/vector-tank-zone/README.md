# Vector Tank Zone

3D vector tank arena inspired by classic tank combat, with procedural Three.js assets, radar, independent turret aiming, pooled particles, objectives, scoring, 25 authored levels, and boss weak points.

## Commands

```sh
npm install
npm run dev
npm run build
npm run smoke
```

Local dev URL, unless Vite picks the next open port:

```txt
http://127.0.0.1:5173/vector-tank-zone/
```

QA URLs:

```txt
http://127.0.0.1:5173/vector-tank-zone/?autostart=1
http://127.0.0.1:5173/vector-tank-zone/?autostart=1&level=5&pixel=1
http://127.0.0.1:5173/vector-tank-zone/?reset=1&unlock=1
```

Production build writes to:

```txt
../../vector-tank-zone
```
